# SopPillars MVP

## Goal

Ship the first playable version of `SopPillars` with:
- arena creation/edit/save
- waiting lobby
- team selection GUI in lobby
- per-player cage spawning
- random loot into inventory
- border shrink
- optional lava rise
- spectators on death
- full arena rollback
- configurable messages/commands

## Implementation Phases

### Phase 1
- project scaffold
- config model
- `/pillars` command tree
- arena repository/files
- editor session flow

### Phase 2
- waiting lobby flow
- join/random matchmaking
- team selection item + GUI
- countdown/start pipeline

### Phase 3
- cages
- kits
- loot pool system
- border/lava runtime

### Phase 4
- death handling
- spectator flow
- isolated chat
- win/end flow

### Phase 5
- rollback
- statistics
- admin polish
