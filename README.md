# SopPillars

`SopPillars` is a `Sop*` minigame plugin for a Lucky Pillars PvP mode.

Current status:
- scaffolded project
- specification captured in [SPEC.md](./SPEC.md)
- MVP broken out in [MVP.md](./MVP.md)

The first implementation target is Paper/Spigot with priority support for:
- `1.16.5`
- `1.21.1`
- `1.21.11`
